What is THE DOM in Javascript?

Every website you visit , consists of an HTML document. We know that at the most basic level, the HTML document gives the structure of the page. While css render the style of the content. So where does the DOM fits in?, DOM stands for Document Object Model that allows javascript to interact with the HTML elements directly.

Dom has a tree like hierachical structure, Whereby HTML elements are represented as nodes. The root of the node starts with the HTML tag which is the parent of all elements inside it, like the body tag. The body tag branches out even further to other child elements such as the h1 or div tag inside it, creating more children. While learning the DOM it's important to understand which of the elements are the "parent" or "children" element. We will dive deeper into that when learning about nodes

Keyword: DOM Parent Children

What are DOM NODES in Javascript?

Every item on the page are represented by Nodeson the page. Which consist of 3 main nodes. The element node , the text node and comment node. The element node is represented by any tags on the html page, whereas the text node is represented by any content written on the page and the comment node refers to any comment we type onto the html document

By representing items in a node , it allows javascript to have another way to interact with the HTML document besides selecting the elements on the page.

Keyword: Element node Text node Comment node
