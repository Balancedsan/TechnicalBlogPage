What is THE DOM in Javascript?

Every website you visit , consists of an HTML document. We know that at the most basic level, the HTML document gives the structure of the page. While css render the style of the content. So where does the DOM fits in?, DOM stands for Document Object Model that allows javascript to interact with the HTML elements directly.

Dom has a tree like hierachical structure, Whereby HTML elements are represented as nodes. The root of the node starts with the HTML tag which is the parent of all elements inside it, like the body tag. The body tag branches out even further to other child elements such as the h1 or div tag inside it, creating more children. While learning the DOM it's important to understand which of the elements are the "parent" or "children" element. We will dive deeper into that when learning about nodes

Keyword: DOM Parent Children

What are DOM NODES in Javascript?

Every item on the page are represented by Nodes on the page. Which consist of 3 main nodes. The element node , the text node and comment node. The element node is represented by any tags on the html page, whereas the text node is represented by any content written on the page and the comment node refers to any comment we type onto the html document

By representing items in a node , it allows javascript to have another way to interact with the HTML document besides selecting the elements on the page. Hoever, the node that we are interested in is the element node

Keyword: Element node Text node Comment node

How to TRAVERSE THE DOM in Javascript?

We explained on the concept of nodes in the previous session , in this session we will talk about traversing the node. We can traverse the node by using the "children" and "parent" keyword as explained briefly in the introduction session. Traversing allow us to indirect target children elements wrapped inside a parent container

Suppose we want to select the body of the document, there are two ways we can do this, the first way document.body which selects the body of the html. By using the "." notation we can access the child of the first element , in this case document refers to the root element and body being the child element of document. Another way we can achieve this is document.children. Which access the child element of document which is the body

Keyword: "." notation document.body document.children

what are DOM EVENTS in Javascript?

Events allow us to trigger responses when an action occur. A real life example will be cooking lunch when we are hungry, whereby being hungry is an event and the response is cooking lunch. Using the idea of events , we want to be able to use events to trigger responses when the user interact with the page. Hence , javascript has event listners to listen for events allow us to control the elements on the page.

An example of an event on a webpage , is a button event whereby a user clicks a button. Using event listners provided by javascript. Javascript will fire off the event depending on what you command it to do. A live example of this is the menu button on the top of the page. If you click on the menu a list of menu option slides down displaying the option on the page. The event being the button , and the response slide down menu option is triggered.

Keyword: Events Responses
